## Reporting a Vulnerability

Since this is a solo project, I handle security reports directly through the repository's Discussions page.

If you find a security-related bug:
Go to the [Discussions](https://github.com/PavanKumarVattikuti/research-agent/discussions) tab.
Open a new discussion in the Bug Reports category.
Provide a clear description of the issue and how to reproduce it.

I'll review it as soon as possible and push a fix in the next update.
